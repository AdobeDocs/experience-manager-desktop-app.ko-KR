# [!DNL Adobe Experience Manager] 데스크톱 앱 정보

[!DNL Adobe Experience Manager] 데스크톱 앱은 [!DNL Experience Manager] DAM 저장소를 데스크톱에 연결하여 리소스를 더 빨리 이용하고 워크플로우를 간소화합니다. 이 작업량 감소는 데스크탑 워크플로우에서 디지털 에셋으로 작업하는 사용자의 효율성을 개선합니다. [!DNL Experience Manager Assets] 내의 자산은 로컬 데스크톱에서 쉽게 액세스할 수 있으며 모든 데스크톱 응용 프로그램에서 사용할 수 있습니다. 원하는 데스크탑 애플리케이션에서 자산을 열고 편집할 수 있습니다. 로컬 편집 내용은 [!DNL Experience Manager] DAM 저장소에서 사용할 수 있습니다. 소스에서 저장소에 추가된 모든 새 에셋을 데스크탑에서도 사용할 수 있습니다.

이러한 통합을 통해 조직의 다양한 역할이 [!DNL Experience Manager Assets]에서 자산을 중앙에서 관리할 수 있습니다. 마케터와 비즈니스 사용자가 브랜딩과 라이센싱을 포함한 다양한 표준 준수를 확보할 수 있습니다. Creative 사용자는 [!DNL Creative Cloud] 및 다른 기본 응용 프로그램의 데스크톱에 있는 자산에 액세스할 수도 있습니다.
